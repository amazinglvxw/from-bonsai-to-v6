Before V6.0 changed anything, it measured everything.

## The Evaluation Framework

Inspired by academic rigor — particularly MemCollab's methodology — we built a 10-metric evaluation framework organized in three tiers.

### Tier 1: Feedback Loop Health

| Metric | Baseline | Target |
|--------|----------|--------|
| M01 Log Density | 0.5/session | ≥2.0 |
| M02 Q-Value Variance | 0 | >0.001 |
| M03 Info Density | 2.2% | ≥20% |

### Tier 2: Memory Efficacy

| Metric | Baseline | Target |
|--------|----------|--------|
| M04 Retrieval Hit Rate | Unknown | ≥60% |
| M05 User Correction Rate | Unknown | ↓30% |
| M06 Context Efficiency | 3.3% | under 15% |
| M07 Retrieval Latency | Unknown | under 2000ms |

### Tier 3: Skill Evolution

| Metric | Baseline | Target |
|--------|----------|--------|
| M08 Skill Reuse Rate | 6.7% | ≥20% |
| M09 Skill Success Rate | 93.3% | ≥85% |
| M10 Active Subsystem Ratio | 33% | ≥50% |

## Golden Dataset: The Control Group

50 representative tasks extracted from real session history. Run against V5.9 baseline, then V6.0. McNemar's test for statistical significance.
